Linux命令案例

Created2019-05-03|Updated2019-10-18|Linux
|Word count:3.3k|Reading time:13min|Post View:

查看套接字

ss -lntp 列出当前开启服务

检查网络连接

ping baidu.com 朝百度发送数据包

telnet 10.0.0.100 TELNET协议的用户界面用来检查网络

系统控制相关

systemctl stop NetworkManager 停用网络管理守护进程

systemctl disable NetworkManager 禁用网络管理守护进程

网卡设置相关

vim /etc/sysconfig/network-scripts/ifcfg-eth0

yum源配置

cd /etc/yum.repos.d/

rm -f ./*

curl -o /etc/yum.repos.d/epel.repo http://mirrors.aliyun.com/repo/epel-7.repo

curl -o /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo

安装常用工具

yun install -y net-tools bash-completion vim lszrz nc nmap ntpdate tree htop glances telnet unzip iftop iotop rsync

更新源

yum repolist 列出仓库可用包

防火墙相关

systemctl stop firewalld.service 关闭防火墙服务

systemctl disable firewalld.service 禁用防火墙服务

vim /etc/selinux/config 配置 selinux

SELINUX=disabled

关机相关

shutdown -h now 立即关机

shutdown -h 10 10分钟后关机

shutdown -c 取消关机或重启计划

poweroff 关机

init 0 0运行级别，关机

重启相关

shutdown -r 10 十分钟后重启

reboot 立刻重启

halt 立刻停止系统，需要人工关闭电源，是 reboot 的链接文件

init 6 切换运行级别到6，6表示重启

注销相关

logout 注销并退出当前用户窗口

exit 注销并退出当前用户窗口

更改网卡默认命名规则

vim /etc/sysconfig/grub

GRUB_CMDLINT_LINUX=’rhgb net.ifnames=0 biosdevname=0 quiet’

网络服务相关

systemctl restart network 重启网络

ip 相关

ip -a 查看所有 ip

ifconfig 查看所有 ip

ip a s eth0 查看第一张网卡

ip address show eth0 查看第一张网卡

ifconfig eth0 查看第一张网卡

主机名

hostname 查看主机名

hostnamectl set-hostname hostname 修改主机名

whoami 查看主机名

uname -m 查看主机名

内核相关

cat /etc/redhat-release 查看内核版本

uname -a 查看内核信息

uname -r 查看 Linux 内核版本

命令行快捷键

ctrl + c cancel 取消当前操作

ctrl + l 清空当前屏幕

ctrl + d 退出当前用户

ctrl + r 查找历史命令 history| grep

ctrl + a 光标移到行首

ctrl + e 光标移到行尾

ctrl + u 把光标到行首的内容删除/剪切

ctrl + y 粘贴

ctrl + k 把光标到行尾的内容删除/剪切

ctrl + → 向右移动一个单词

ctrl + ← 向左移动一个单词

ctrl + s 锁屏

ctrl + q 解锁

ctrl + w 删除当前位置至上一个空格的字符

cd 相关

cd ~ 切换到当前目录的家目录

cd - 切换到上一次所在位置的目录下

cd / 切换到根目录

tree 相关

-d 只显示目录

-L 遍历目录的最大层数，后面加大于0的正整数

-F 在条幕后加上文件类型的指示符号{*,/,=,@,|,其中一个}

tree -L 1 -F /boot/ 使用-F 参数会在目录后添加’/‘方便区分目录

mkdir 相关

mkdir test -v 穿件目录时打印相关信息

mkdir linux/test -p 父目录不存在时一起创建，递归创建目录

mkdir {oldboy,oldgirl,oldman} 花括号里面的同时创建

mkdir oldboy{a..z} 创建 oldboya-oldboyz的目录，创建不连续的目录时用逗号隔开

touch 相关

{} 生成序列

{1..10}

{01..10}

{a..z}

ls 相关

-a 显示指定目录下的所有子目录与文件，包括隐藏文件

-l 以列表方式显示文件的详细信息 ls -l=ll

-h 配合 -l 以人性化显示文件大小

-t 根据最后修改时间排序，默认是以文件名排序，通常与-l 连用

-F 在条目后面加上文件类型的指示符号

-d 显示目录本身的信息，而不是显示目录的内容

-r 逆序，倒序排序

ls -lt 按照时间进行排序

ls -lrt 找出最新的文件

-i 显示索引节点信息

cp 相关

-r 递归复制目录

-p 复制的时候保持属性不变

-d 复制的时候保持软链接

-a == -pdr

-t 将所有源参数复制到目录中

vim 相关

底行模式

:wq 保存并退出

:x 保存退出

:q 退出

:q! 强制退出不保存

:set nu 显示行号

:set nonu 取消显示行号

:set list 给每行的最后一个字符加上标记

:set nolist 取消标记

:noh 取消高亮显示

:%s###g 替换内容

编辑模式

a 在光标之后开始插入数据

i 在光标之前开始插入数据

A 在光标所在行末尾开始插入数据

I 在光标所在行首开始插入数据

o 在当期行下面插入一行并进入编辑魔术

O 在当前行上面插入一行并进入编辑模式

C 删除光标所在位置到行尾的内容并进入编辑模式

批量操作

批量删除：ctrl+v 选择 d 删除

批量增加：ctrl+v 选择 shift+i编辑 ESC

删除多行：shift+v 选择 d

快捷键

gg 移到文件第一行

G 移到文件最后一行

100gg 移到文件第100行

^ 把光标移到行首

$ 把光标移到行尾

D 删除光标所在位置到行尾的内容

ZZ 快速保存并退出

yy 复制光标所在行

nyy 复制多行

p 粘贴

np 多次粘贴

dd 删除/剪切一行

ndd 删除/剪切多行

dG 删除当前行到文件的末尾

u 撤销(命令行模式下)

r 替换光标所在字符

x 删除光标所在字符

cat 相关

-b 显示行编号（忽略空行）

-n 显示所有行编号（包括空格）

-A 在每行的最后一个字符加上字符

文件名 清空文件内容

cat >文件<<EOF 以 EOF 开头以 EOF 结尾

EOF 可以是任何字母或数字

cat>>文件<<EOF

表示写入新内容，即使之前有内容也会覆盖

表示在文件的末尾追加新内容，之前的内容不覆盖

head 相关

head显示文件的前几行（默认显示10行）

-n5 显示前5行

tail 相关

显示文件的最后几行

-n5 显示文件最后5行

-f –follow 显示文件实时更新（查看日志）

-F –follow –retry 如果文件不存在会不断进行重试

网络测试相关

ping 探测指定 ip 或域名的网络情况

telnet 远程管理与端口探测命令

traceroute 路由跟踪命令

wget 下载命令

tcpdump -i eth0 -nnx port 21

-i 指定网卡接口

-nn 将数据包中的域名与服务转为 ip 和端口

-X 以十六进制和 ASCII 码显示数据包内容

port 指定监听的端口

find 相关

find / -type f -name ‘*yum’ 查找文件类型为文件以 yum 结尾的文件

find / -maxdepth 2 -type f -iname ‘*.txt’ | xargs ls -l 查找最大深度为2文件类型为文件以 txt 结尾且忽略大小写的文件

find / -maxdepth 4 -type f -iname ‘*.txt’ | xargs ls -l 查找最大深度为4普通文件忽略大小写以txt结尾的文件

find ./ -type f -name ‘*.txt’ |xargs cp -t /tmp 在当前文件夹下查找普通文件以 txt 结尾的文件并拷贝到 tmp 目录下

find ./ -type f -namd ‘*.txt’ |xargs -i cp {} /tmp/a 在当前目录下查找以 txt 结尾的普通文件并拷贝到/tmp/a目录下

cp $(find ./ -type f -name ‘*.txt’) /tmp/a 先执行括号里面的之后拷贝

find ./ -type f -name ‘*.txt’ -exec cp {} /tmp/c ;

find / -type f -size +100k -mtime -7 -name “*.txt” |xargs ls -lh 在根目录下查找文件大小100k 以上以 txt 结尾的且在7天之内修改的普通文件

find / -type f -size +100k -mtime +7 -name “*.txt” |xargs ls -lh

find ./ -type f -name “*.sh” |xargs sed “s#oldgirl#python#g” 在当前目录下找到以 sh 结尾的普通文件并把文件里的 oldgirl 替换为 python

find ./ -type f -name ‘*.sh’ | xargs sed -i ‘s#oldgirl#python#g’

标准输入输出错误

标准输入：代码为0，使用<或<<

标准输出：代码为1，使用>或>>

标准错误输出：代码为2，使用2>或2>>

2>&1 错误信息和正确信息放在一起

将标准输入转换为命令行参数

seq 10 > ett.txt

xargs -n5 < ett.txt

-n 分组，主要配合 find 命令使用

tar 相关

tar zcvf etc.tar.gz /etc 将/etc打包为etc.tar.gz

tar tf etc.tar.gz 不解压列出压缩包内容

tar zcf etc1.tar.gz –exclude=/etc/sysconfig /etc 除了 sysconfig 文件其他打包

tar xf etc.tar.gz 解压

tar xf etc.tar.gz -C /opt/ 指定解压路径为/opt/

zip 相关

同步网络时间

ntpdate ntp1.aliyun.com

时间相关

date +%F 2019-04-10

date +%Y-%m-%d 2019-04-10

date +%y-%m-%d 19-04-10

date +%T 16:33:37

date +%H:%M:%S 16:34:05

date +%w 3(周三)

date -d “-1 day” +%F 2019-04-11

date +%F-%T 2019-05-12-11:00:05

tr 相关

替换或删除字符

tr 与 < 连用

-d 删除字符

tr ‘a-z’ ‘A-Z’ < 文件名 把文件里面的 a-z字母替换成大写字母

字符集相关

localectl set-locale LANG=zh_CN.UTF-8 设置字符集为中文

source /etc/locale.conf

echo $LANG 查看字符集

查看用户信息

echo $UID 查看当前登录用户 uid

cat /etc/passwd 查看所有用户信息

cat /etc/group 查看组信息

tail /etc/passwd 查看最后十行用户信息

cat /etc/shadow 查看用户密码（加密）

添加用户

useradd -u 666 web 指定 uid 创建用户

groupadd -g 888 test 指定 gid 创建组

useradd -u 777 -g 888 test 指定 uid 和 gid 创建用户

useraddd -u 456 -g 888 -s /sbin/nologin -M xuni 指定 uid 和 gid 指定登陆 shell 以及不创建家目录创建用户 -s 和-M 需搭配使用

修改密码

passwd 修改当前登录用户密码

passwd username 修改用户密码

echo ‘1’ | passwd –stdin username 免交互方式改变密码

whoami 显示当前登录用户名

CTRL+d 登出当前用户

id username 查看用户 uid、gid、groups

修改用户组

usermod -u 999 username 修改已有用户 uid

usermod - g 999 username 修改已有用户 gid

usermod -s /sbin/nologin username 修改已有用户不能登陆

groupdel test 删除用户组

删除用户

userdel -r username 删除用户及用户家目录

groupdel groupname 删除用户组

时间相关

stat filename 查看文件详细属性

索引节点

df -h 查看分区信息

df -i 查看分区 inode 数量使用情况

ll -i 显示 inode 号

链接

ln (source file) (destination file) 创建硬链接 类比超市入口

ln -s (source file) (destination file) 创建软连接，相当于一个快捷方式

创建测试文件

dd if=/dev/zero of=/data/test/test.txt bs=10M count=1000 创建一个10000M 的文件

du -sh /*|grep G 查看根目录下的文件大小

寻找占用 inode 大的文件夹

find / -type d -size +1M 查找根目录下面大小1M 以上的目录

find / -type d -size 1M |xargs ls -ldh 寻找占用小文件过多的目录

ll | wc -l 统计当前目录下面的文件数量

批量删除文件

find ./ -type f -name “*.txt” | xargs rm -f 批量删除以 txt 结尾的文件

rm -rf ./odldboy0? 删除当前目录下结尾为任意一个字符的文件

mkdir odldboy{00..10} 批量创建目录

rm -rf ./oldboy0[12]test []表示或，删除

rm -rf ./oldboy[!3]test !表示非，删除除3之外的目录

生成随机字符

tr -cd ‘a-z0-9’ < /dev/urandom | head -c 20 生成20位随机字符

echo 相关

echo $PATH 先执行反引号里面的，之后打印出来

echo $PATH; date 命令分隔符，执行两条命令换行打印

echo -n $PATH; echo oldboy 一行输出

echo 1234567 | sed -r ‘s#(3)(4)#<1><2> #g’ 替换为12<3><4> 567

wc 相关

ll | wc -l 计算当前目录下面的文件数量

sed 相关

ifconfig eth0 | sed -nr ‘2s#(.)t (.) n(.*)#2#gp’ 过滤出 ip 地址

ip a s eth0 |sed -nr ‘3s#(.)t (.)/(.*)$#2#gp’ 过滤出 ip 地址

grep 相关

grep -o ‘[a-Z]’ oldboy.log |sort |uniq -c |sort -nr 查找文件中的字母并进行倒叙排序

添加克隆机后网卡设置

hostnamectl set-hostname 主机名

sed -i ‘s#100#8#g’ /etc/sysconfig/netword-scripts/ifcfg-eth[01] 设置内网 ip 和外网 ip

systemctl restart netword 重启网络服务

bash 重新登录 bash

关闭邮件通知

echo “unset MAILCHECK” >> /etc/profile

source /etc/profile

scp 相关

scp -r -P 520 ~/.vimrc root@192.168.11.171:/home/ 本地局域网传输文件到其他局域网里的虚拟机

scp -P 522 docker_python3.6.tar.gz root@10.0.0.11:/home 从本机传到本机虚拟机上
