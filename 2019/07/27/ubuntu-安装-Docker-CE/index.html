Ubuntu18.04 安装 Docker CE

  1. 添加使用 https 传输的软件包和 CA证书
  1. 添加软件源的 GPG 密钥
  1. 添加 docker 软件源
  1. 安装 Docker CE
  1. 或者使用脚本自动安装
  1. 启动 docker
  1. 建立 docker 用户组
  1. 将当前用户加入 docker 组

退出终端并重新登录或者重启 docker。

  1. 测试使用不使用 sudo 能否执行 docker 命令

如果提示

则修改/var/run/docker.sock权限
