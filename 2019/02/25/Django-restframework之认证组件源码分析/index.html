三 自定义认证组件

了解了 restframework 的认证流程，对于需要自定义认证组件其实很明了，就是自定义认证类，重写 authenticate 方法。不过这还不行，我们还需要进行一下配置。

首先不管 cookie 或 session 都是为了校验用户的，那么在这里我们可以使用一个随机字符串（加密后的），当客户端朝服务端发送请求时会携带该值，之后进行反解用来对比。
