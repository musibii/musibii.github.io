通过字符串来操作类或对象的属性

发表于 2018-10-25   |   更新于 2019-10-18 | 分类于 Python | | 阅读数
字数统计: 796word | 阅读时长 ≈ 3min

通过字符串来查看获取修改删除类或对象的属性的使用场景为获取用户的输入,因为得到的值都是字符串类型,那么就可以直接通过字符串来操作属性而不用把字符串修改为可以通过句点法访问的形式了.

那么有四个方法可以通过字符串来操作属性:

  1. hasattr:是否具有该属性,返回布尔值
  2. getattr:获取属性值,没有该属性的话会报错,设置一个默认值 None
  3. setattr:修改属性值, 有就修改,没有就添加
  4. delattr:删除属性,有就删除,没有会报错,应该加个判断
